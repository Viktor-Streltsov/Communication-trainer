# Communication Trainer

Приложение для тренировки навыков общения с помощью AI.

## Структура

- `mobile/` — мобильное приложение (React + Capacitor)
- `backend/` — серверная часть (FastAPI)
- `docs/` — документация и план проекта

## Описание

Communication Trainer помогает пользователям отрабатывать разговорные сценарии,
получать обратную связь и прокачивать коммуникативные навыки в безопасной среде.
